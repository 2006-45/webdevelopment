World Wide Web (WWW)

The World Wide Web (WWW) is an information system where documents and resources are linked using hypertext and accessed via the Internet. Invented by Tim Berners-Lee in 1989 at CERN, it became publicly available in 1991.

The Web is built on three core technologies: HTML (for structure), HTTP (for communication), and URL (for addressing). It is not the same as the Internet — the Internet is the underlying global network, while the WWW is the service running on top of it.

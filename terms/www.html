World Wide Web (WWW)

The World Wide Web (WWW) is a wide-area hypermedia information retrieval initiative aiming to give universal access to a large array of documents.

Key Characteristics:

  • Wide-area: Spans the entire globe
  • Hypermedia: Contains text, images, sound, video and hyperlinks
  • Information retrieval: Easy access via web browsers
  • Universal access: Platform independent

Important: The WWW is a service that runs on the Internet, not the Internet itself.
